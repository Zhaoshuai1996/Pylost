PyLOSt.data_in.msi.opd_to_h5 module¶

Created Mar 21,2018

Class to convert the *.opd files generated by Veeco MSI instrument to *.h5 files. A measurement has the following files 1. Raw patches from different scans (n scans x m subapertures) 2. Header metadata
