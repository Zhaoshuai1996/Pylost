PyLOSt.data_in.sharper.loadHasData module¶

Created on Nov 10, 2017

Method to load sharper measurements and convert to h5 input:

in_folder: folder containing .has files

output:
Single HDF5 file containing

  1. header meta data (exp conditions, instrument params …)

  2. rawdata (slopes X/Y, height), all patches, all scans

  3. stitching versions (a new version added when stitched by new algorithm or different parameters)

@author: ADAPA
