class PyLOSt.ui.hdf5widget_custom.Hdf5TreeViewExample(filenames=None)[source]¶

Bases: PyQt5.QtWidgets.QMainWindow

This window show an example of use of a Hdf5TreeView.

The tree is initialized with a list of filenames. A panel allow to play with internal property configuration of the widget, and a text screen allow to display events.
