pylost_widgets.util.cm_scaling module¶

The ‘ColorScale’ class is created by Francois Perrin (ESRF) for better visualization of colors for 2d height/slope errors. It implements color schema based on ‘turbo’, with added color scaling based on peak to valley / standard deviation / mean absolute deviation etc.

class pylost_widgets.util.cm_scaling.ColorScale(surface, base='ra', *params)[source]¶

Bases: object

Class to manage colormap scaling based on ‘turbo’ maps adjusted by data rms or pv or mad etc. parameters

base: ‘ra’, ‘rq’, ‘pv’, ‘fixed’ params: list

if stats based: ‘factor’ as stretching factor if manual scale: ‘zmin, zmax, z_lo, z_hi’

or ‘default’ as half-scale default values

Stack dictionary objects (e.g. while loading sequence of files). Only numpy arrays, lists or tuples are stacked.

Parameters

  • a (dict) – Input dictionary

  • b (dict) – Output stacked dictionary

  • a_f (dict) – Parent input dictionary. By default parent is the same as input dictionary. If dict elements are sub directories, stack_dict is called iteratively, and a_f transfers link to parent dictionary

  • b_f (dict) – Parent output stacked dictionary

  • start_pos_keys (list[str]) – Start position keys in the dictionary. When stacking two images of different sizes from same instrument, start position of the image relative to detector is used for the merger.

  • stack_selected_keys (list[str]) – Stack selected keys in the dictionary, even if the values are not numpy arrays or lists. E.g. stack motor position, even though it may be a float for a single image

  • cam_size_keys (list[str]) – Camera size keys in the dictionary. If provided and the images match the camera size, start_pos_keys are ignored.

  • merge_selected_keys (list[str]) – Merge selected keys instead of stack into list, e.g. stitching LTP where a scan is a sequence of subscans with overlaps and the final list of positions is merger of subscan positions

  • start_pos (list[float]) – Updated start position of Output stacked arrays

Returns

start positions keys, current start position

Return type

list[str], list[float]
