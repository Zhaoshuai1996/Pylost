Save data widget guide

Save orange pylost data to pickle or hdf5 formats. Pickle format saves all data in native & custom python datatypes, whereas hdf5 is limited to few types like numpy arrays, strings, floats etc. Pickle can be directly reloaded within orange pylost without any modifications and so it is extremely helpful for saving intermediate processed data. However HDF5 is quite useful across platforms and pickle may not correctly load in another python environment.

This widget also has option to save heights data as MetroPro dat format, so that it can be viewed through MetroPro. Similar writers can be easily implemented by users and can be integrated into orange pylost.

The widget has basic saving where all data directly dumped to a file. It also has advanced saving where partial data can be dumped to file, file dataset names can be modified or datasets deleted, file hdf5 structure can be modified and so on.
