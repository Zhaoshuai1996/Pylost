Stitch params widget guide

This widget allows to select a stitching algorithm from predefined algorithms and additionally select different algorithm options. An example to 'global optimization' algorithm and its options are presented in below image.

Stitching options are common to all whereas algorithm options are specific to the selected algorithm. A constant stitching step can be used with 'stitch step x', 'stitch step y' instead of recorded motor positions, by checking corresponding checkbox as shown in above image.
