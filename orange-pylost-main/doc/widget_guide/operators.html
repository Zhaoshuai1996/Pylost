Operator widgets guide

Operators like Add/average, Subtract, Multiply, Divide can be used with multiple input data. The input datasets should broadcastable if not of same shape. The operations are applied in sequence, i.e. for subtraction  A - B - C - D etc. where A,B,C,D are the inputs added in that sequence.

Add widget has a checkbox to scale the output by number of inputs, i.e. to calculate average.
