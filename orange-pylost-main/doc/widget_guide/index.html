Introduction

Guide for all the widgets created for orange pylost is presented here.

This guide has multimedia images and videos, which may be rendered more accurately in a native browser rather than Help widget browser.

The orange-pylost widgets are developed for surface metrology data analysis of synchrotron X-ray mirrors. The widgets can be categorized into the following.

  • Data loading and saving:
    • Data (File): Load a single file or sequence of files. Typically used to load raw measuremnt data of metrology instruments
    • Data (H5): Load measurement data form PyLOSt custom format HDF5 file
    • Data (scans): Load repeated scans in a folder
    • Save Data: Save processed data in pickle or hdf5 formats
  • Visualize data
    • Visualize: Visualize N-dimensional slope / height data
    • Visualize Compare 2D/1D: Compae 2D/1D plots of slope / height data
  • Stitching widgets
    • Stitch Parameters: Select stitching algorithm and options, and stitch subapertures.
    • Visualize Stitch Results: Display results such as stiched heights / slopes, extracted reference etc.
  • Operators
    • Basic operators: Add, subtract, multiply or divide multiple datasets
    • Merge data: Merge/join multiple datasets along a new / first / last axis
  • Miscellineous
    • Mask: Apply mask to subapertures or stitched images
    • Fit: Fit polynomial or ellipse and subtract best fit
    • Flip data: Flip any input data along X (last axis) and/or along Y (last - 1 axis)
    • Gravity correction: Add (or subtract) gravity profile from 1D/2D data
    • Integrate slopes: Integrate 1D slopes or 2D slopes_x, slopes_y data
    • Interpolate: Interpolate data nans or new pixel size or subpixel offsets
    • Optimize XY: Optimize subaperture shift in XY plane for lowest overlap errors
    • Rotate: Rotate datasets by given angle
    • Threshold: Apply min/max threshold on slopes / height data
    • Select subapertures: Exclude some of the loaded subapertures and ouput rest
    • Average scans: Average all or selected scans, only works if multiple scans present in data
    • Average subapertures: Average l x m x n subaperture sequence to get m x n image
    • Filter: Apply a Gaussian low-pass / high-pass filter
    • Bin data: Bin data along X and/or Y axis
