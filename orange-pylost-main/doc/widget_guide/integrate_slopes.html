Integrate slopes widget guide

Integrate 1D slopes or 2D tangential and sagittal slopes. For 2D integration both tangential and sagittal slopes are required. The input arrays can N-dimensional but integration is applied to only last two dimensions.

1D integration is done using cumulative trapezoidal method, whereas 2D integration uses Sylvester or Frankot Chellappa (default) methods.

references:

Frankot, R. T., & Chellappa, R. (1988). A method for enforcing integrability in shape from shading algorithms. IEEE Transactions on pattern analysis and machine intelligence, 10(4), 439-451.

Kirsten, G. P. (2018). Comparison of methods for solving Sylvester systems (Doctoral dissertation, Stellenbosch: Stellenbosch University).

Both the 2D integration algorithms utilitze software code implemented at ESRF for 'Speckle Wavefront Reconstruction Package'.
