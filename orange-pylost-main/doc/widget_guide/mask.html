Mask widget guide

Draw mask and apply to (a) single image, (b) one or all images in a sequence, (c) one image in a subaperture sequence in all repated scans and (d) all subapertures in one scan.

Mask can be drawn on figure by drawing a rectangle, or drawn with given center and size values. Center of mask can be relative to current image size or relative to detector size. Mask can be drawn in pixels or pixel length units. Mask can be moved with arrow keys.
