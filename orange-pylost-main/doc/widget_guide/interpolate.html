Interpolate widget guide

Interpolate data with one of the following ways. If the data is of type MetrologyData, detector dimesnions are taken from its attributes, else last two dimensions are used for N-dimensional data. Interpolation is done only in either 1D or 2D, any higher dimensions are looped over.

  • Interpolate nan and inf values.
  • Interpolate to a new pixel size in X and/or Y
  • Interpolate subpixel offsets between subapertures, due to non-integer-pixel translations of motors

Interpolation uses interp1d or interp2d functions from scipy.interpolate module.
