Data (File) widget guide

This widget is used to load a single or a sequence of files. It allows loading of many file formats some of which are as following.

  • Raw instrument data formats such as
    • Zygo Fizeau 'dat' / 'datx'
    • Veeco MSI 'OPD'
    • SHARPeR 'has'
  • Simple ascii formats such as
    • ESRF LTP formats 'slp', 'slp2'
    • Simple 'txt' files as follows
  • Pickle (pkl) and HDF5 formats. Pickle can store and reload in native python data types.

The output of the widget will be a dictionary object, typically containing default data such as heights, slopes_x, slopes_y for data analysis with other orange pylost widgets
